# EID al-Fitr CLI Quiz Game 🎉

A fun and interactive command-line quiz game to test your knowledge about **EID al-Fitr**! Answer all questions correctly to win the game, but beware — a wrong answer will end the game immediately.  

Built with Node.js using **Inquirer**, **Chalk**, **Chalk-Animation**, **Gradient-String**, **Figlet**, and **Nanospinner**.

---

## Features

- Interactive CLI game with colorful animations 🎨
- 10 questions about EID al-Fitr 🕌
- Immediate feedback on each answer
- Fun winner animation when you complete the quiz 🎊

---

## Getting Started

### Requirements

- [Node.js](https://nodejs.org/) (v14 or later)
- npm (comes with Node.js)

### Run via npx

You don’t need to install anything manually, just run:

```bash
npx eid-game
